# Types and Variables

Simple example of how variables are defined and there types explicitly:

```rust
fn main() {
  let my_var:u32 = 1000;
}
```

Simple example of how variables are defined with types implicitly:

```rust
fn main() {
  let my_var = 1000; // assumed to be i32
}
```
